## Technische SEO-Checkliste

- [x] `robots.txt` – erlaubt alle Crawler
- [ ] `sitemap.xml` – alle Seiten mit `lastmod` + `priority`
- [ ] Canonical URLs auf jeder Seite
- [ ] `hreflang` für DE/EN (optional, da Mehrsprachigkeit vorhanden)
- [ ] Lazy Loading für alle Bilder
- [ ] Alt-Texte für alle Bilder mit Keywords
- [ ] Core Web Vitals optimieren (LCP, FID, CLS)
- [ ] `rel="noopener noreferrer"` auf externen Links

## Content-Strategie (Semantische Tiefe)

**Immer im Hintergrund erwähnen:**
- Halal, vegan, vegetarisch, frische Zutaten
- Superfood, Teff, glutenfrei, gesund essen
- Gewürze: Berbere, Mitmita, Niter Kibbeh
- Essen mit Fingern, Gemeinschaftsgericht, Injera als Teller
- Äthiopische Kultur, Kaffeezeremonie, Fastengerichte
- Salzburg + Äthiopien, Österreich + Afrika
- Besonderes Erlebnis, einzigartiges Restaurant

**Für KI-Suchen (ChatGPT, Perplexity, Google SGE):**
Natürliche Frage-Antwort-Strukturen einbauen:
- "Was ist Injera?" → Antwort im Text
- "Ist das Restaurant halal?" → Antwort sichtbar
- "Kann man online bestellen?" → Antwort + Link
